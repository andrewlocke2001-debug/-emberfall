# EMBERFALL — Inspiration Map

"All the best in one" only works if each steal is deliberate, adapted, and sequenced. This table is the contract. Cells are terse on purpose — full designs live in `docs/GDD.md`.

## What we're taking

| Source | Steal | Why it's best-in-class | Our version | Phase |
|---|---|---|---|---|
| OSRS | 600ms tick combat | netcode-cheap, skill-expressive | identical cadence | P2 |
| OSRS | skill-by-doing + total level | clearest progression ever | 8 skills → expand | P2/P4 |
| OSRS | hiscores | infinite long-game motivation | public web boards | P6 |
| OSRS | Slayer | best retention per dev-hour | Hunts system | P10 |
| OSRS | Ironman/Hardcore | self-made prestige modes | flagged modes | P10 |
| OSRS | AFK-friendly skilling | respects real life | gather rhythms | P4 |
| RuneScape | one world, one economy | reputation + market depth | districts, no shards | P0 |
| RuneScape | Grand Exchange | async order-book market | the Exchange | P8 |
| RuneScape | Wilderness | opt-in risk = legend factory | the Ashreach | P9 |
| RuneScape | quests as set pieces | quests people remember | 5 spotlight quests | P5 |
| WoW | quest-hub flow | best onboarding ever | starter arc | P5 |
| WoW | dungeons + boss mechanics | group content gold standard | Cinder Depths | P7 |
| WoW | raids + lockouts | aspirational peak | 8-man raid | P12 |

## What we're deliberately NOT taking

| Source | Feature | Why cut |
|---|---|---|
| WoW | gear-treadmill tiers | endless inflation, solo-dev death |
| WoW | hard trinity roles | shrinks build space, queue hell |
| WoW | flying | trivializes world |
| Tibia/old UO | open PvP everywhere | griefing kills small games |
| Lost Ark etc. | P2W/lockboxes | never |
| FFXIV/WoW | housing (at launch) | DB + content nightmare; v2 maybe |
| EVE | full sandbox scam-economy | wrong tone for this game |
| All | multiple servers | splits tiny playerbase; districts instead |
